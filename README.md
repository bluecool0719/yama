# Digital Yama

통합문제집 기반 CBT 형식의 웹입니다.

## 구성

- 과목 선택 화면 제공
- 과목별 문제 풀이 화면 제공
- 한 페이지에 10문제씩 표시
- 통합문제집에서 추출한 문제 데이터와 이미지 사용

## 파일

- `index.html`: 메인 진입 파일
- `app.js`: 화면 렌더링 및 상호작용
- `styles.css`: 레이아웃 및 스타일
- `data.js`: 문제 데이터
- `assets/images/docx/`: 원본 문서에서 추출한 이미지

## 실행

정적 파일 프로젝트이므로 `index.html`을 브라우저에서 열면 바로 실행할 수 있습니다.

## 참고

- 원본 문제집: `통합문제집.docx`
- 데이터 재생성 스크립트: `scripts/extract-docx.ps1`
